# PyTetris
*Tetris on Python with modern system and TGM grading*
___
## Requirement
* Python 3.13.2 -
* Python arcade 3.1.0
* Windows, mac or Linux


## Control
* **Left Arrow / Right Arrow** : Left and Right Move
* **Up Arrow** : Hard Drop
* **Down Arrow** : Soft Drop
* **Right Shift** : Sonic Drop
* **Z** : Counter Clockwise Turn
* **X** : Clockwise Turn
* **C** : 180 Turn
* **A** : Hold Piece
* **Escape** : Pause
* **R** : Reset
